analysis package
================

Provides analysis of dataflow actors and networks.

Subpackages:
analysis.actor      Analysis at actor level.
analysis.air        AIR=Analysis IR, interfaces that specify this package's requirements on the IR.
analysis.iradapter  Caltoopia IR wrappers and implementations of the Analysis IR.
analysis.test       Mechanisms to test the analysis package from the command-line (outside eclipse). 
analysis.util       Various utilities and abstract data types, used by the analysis.
